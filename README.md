# Trello Clone App

## Project setup

Setup local env file:
```sh
cat .env.example > .env
```

Now edit `.env`, and set your tokens:
```sh
REACT_APP_TRELLO_API_BASE_URL=https://api.trello.com/1
REACT_APP_TRELLO_API_KEY=********
```

Install node modules:
```sh
npm i
```

Start dev server:
```sh
npm start
```

Build for production:
```
npm run build
```
